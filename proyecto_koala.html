Koala Chik

KoalaChik es un proyecto desarrollado para un cliente propietario de una tienda de ropa para señoritas. El cliente puso como requerimiento principal que el sitio debia ser en Wordpress y WooComerce.

En este proyecto estuve encargado de personalizar la plantilla del sitio.

Para este proyecto se desplegó wordpress en un servidor y se configuro certificados SSL.
