Flex Page

Flex page es una plantilla responsive design hecha en html y css, con el obejetivo de poner en practica mis conocimientos de maquetacion de paginas web y CSS.

En este proyecto usé la tecnología flexbox y media querys. Este proyecto se adapta a distintos tipos de pantallas.

Además este proyecto esta a disposicion de cualquier persona para su libre uso.
