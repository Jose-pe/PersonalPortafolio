Musión Huascaran

Proyecto para Misión Huascaran, sitio web hecho con html, css y js, cuenta con una tienda de entradas para un evento benéfico.

En este proyecto usé la tecnología flexbox y media querys. Este proyecto se adapta a distintos tipos de pantallas e incluye de pagos con tarjetas VISA.
